# 🃏 Face-Up Playing Cards Portfolio

A premium, high-performance portfolio website built with React and TypeScript, featuring a unique "Face-Up" playing card aesthetic on a casino felt table.

## 🚀 Live Demo
**View the live site here: [https://cardsthemedportfolio.vercel.app/](https://cardsthemedportfolio.vercel.app/)**

## ✨ Features
- **Zero-Interaction UX**: All professional information is permanently visible and beautifully laid out across a virtual casino table.
- **Cinematic Animations**: Cards animate ("deal") onto the table upon page load with a realistic arc.
- **Tactile UI**: Interactive hover-lift effects on cards for a physical feel.
- **Responsive Design**: Elegant spread on desktop and structured stacking for mobile devices.
- **Tech Stack**: React 18, TypeScript, Vite, CSS (Template Strings).
- **Communication**: Integrated contact form powered by Web3Forms.

## 🛠️ Tech & Tools
- **Framework**: [React](https://reactjs.org/)
- **Styling**: Vanilla CSS with template literals for extreme speed.
- **Deployment**: [Vercel](https://vercel.com/)
- **Forms**: [Web3Forms](https://web3forms.com/)
